# My-Assignments-on-Data-Analytics-Work

This repository contains my hands-on assignments completed as part of my Data Analytics learning journey. It demonstrates my practical skills in **SQL, Microsoft Excel, Power BI, and Tableau**, showcasing my ability to clean, analyze, visualize, and interpret data. Each assignment is designed to solve real-world business problems using industry-standard tools and best practices.

The **SQL Assignment Solutions** include queries covering data retrieval, filtering, sorting, joins, aggregate functions, subqueries, window functions, Common Table Expressions (CTEs), and data manipulation techniques. These solutions demonstrate my understanding of relational databases, SQL query optimization, and analytical problem-solving skills commonly required for Data Analyst roles.

The **Excel Assignments** focus on data cleaning, formatting, formulas, functions, PivotTables, charts, conditional formatting, and dashboard creation. The **Power BI Assignment** showcases interactive dashboards with data modeling, Power Query transformations, DAX measures, KPIs, slicers, and insightful visualizations that support business decision-making. These projects highlight my ability to transform raw data into meaningful reports and actionable insights.

The **Tableau Assignment** demonstrates my skills in building interactive dashboards and visual analytics using calculated fields, filters, parameters, hierarchies, maps, and storytelling features. Overall, this repository reflects my proficiency in key Data Analytics tools and my commitment to continuous learning, making it a strong portfolio for internships and entry-level Data Analyst opportunities.
